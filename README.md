# Agenda de Contactos en C

Agenda interactiva por consola desarrollada en C como segundo proyecto 
del portafolio. Desarrollada durante el primer año de la carrera 
de Desarrollo de Software.

## Funcionalidades
- Agregar contactos (nombre, teléfono y email)
- Mostrar todos los contactos
- Buscar contacto por nombre
- Eliminar contacto por nombre
- Persistencia de datos en archivo contactos.txt
  
## Conceptos aplicados
- Structs para agrupar datos relacionados
- Arrays de structs
- Manejo de strings con string.h (strcmp)
- Menú interactivo con while y switch

## Compilar y ejecutar
gcc agenda.c -o agenda
./agenda
